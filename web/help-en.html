SpritzPlanning guide

Spritz-themed planning poker for agile teams. No account needed — nickname and room code are enough to start.

Open the bar In-app guide

Roles

The bartender facilitates: starts votes, reveals cards, manages the order menu. Guests vote with cards. Observers follow without voting.

Session flow

  1. Open or join a room
  2. Add orders to the menu (backlog)
  3. Start voting for each story
  4. Reveal votes and agree on the estimate
  5. Export the end-of-session report

Key features

  • Backlog import — from Tools menu in the room
  • Auto-reveal — deck settings when everyone has voted
  • CSV/Markdown report — summary icon in the room
  • Resume session — from home if you played recently

Frequently asked questions

Nickname already taken

That nickname is active in the room. Leave first or wait ~2 minutes if the client disconnected.

Rejoining the same room

Use the same nickname and room code. The app offers to resume if you left recently.

PIN not accepted

Ask the bartender for the PIN. It must be 4–6 numeric digits.

Observer mode

Check «Observer only» when joining. You cannot vote but you can see reveals and reports.
